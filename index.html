Equivalent String

Two string are said to be equivalent if they have same set of character and for each character if difference in frequency is not more than 3 .Then , Two string are said to be equivalent to each other.

Logic

Here, to check two string we are going to make two array for storing frequency of each character .we are going to iterate for each charcter in the string and store in vector.the we will check in both araay if both have frequency not differ by 3 and return our decision on this basis.

Code

bool checkAlmostEquivalent(string word1, string word2) {
vector w1,w2;
int i,j,c,n=word1.length();
char ch;
for(i=0;i<26;i++){
w1.push_back(0);
w2.push_back(0);
}
j=0;
for(ch='a';ch<='z';++ch){
i=0;
c=0;
while(word1[i]!='\0'){
if(word1[i]==ch){
c++;
}
i++;
}
w1[j]=c;
i=0;
c=0;
while(word2[i]!='\0'){
if(word2[i]==ch){
c++;
}
i++;
}
w2[j]=c;
j++;
}
for(i=0;i<26;i++){
c=(w1[i]-w2[i]);
if(c<0){
c=c*(-1);
}
if(c>3){
return false;
}
}
return true;
}
